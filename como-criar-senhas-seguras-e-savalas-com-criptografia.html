Como criar senhas seguras e salva-las com criptografia?

Senhas grandes, que tenham números, letras maiúsculas, minúsculas e símbolos são muito importantes para manter sua proteção online.
Mas é bem chato precisar ficar memorizando todas as senhas com todas essas variações para cada serviço. Para isso, existe o Keepass.
Keepass é um software que além de criar senhas seguras, ele consegue armazena-las com criptografia, onde a única forma de obter o banco de dados é com a senha mestra que você mesmo cria ao iniciar o programa, esse arquivo gerado onde ele guarda suas senhas é extremamente pequeno, então você pode colocar ele em qualquer pendrive, deixar em armazenamento nuvem sem problemas que praticamente não ocupa espaço. (recomendo que crie backups online ou físicos(pendrive ou hd) para nunca perder suas senhas)

  • Como usar o programa?

É bem simples utilizar o programa, primeiramente você precisa fazer o download do Keepass, e para isso, acesse o site oficial do projeto aqui.


Windows


No canto esquerdo do site tem uma lista, procure por Downloads e clique em Download Now, depois de baixar, basta executar e software e seguir com a instalação. Ela está disponível em pt-br e é bem fácil.

Linux

Para instalar o Keepass no Linux também é igualmente simples. Ele está disponível por padrão nos repositórios AUR, RPM e DEB, basta iniciar seu terminal e colar o comando respecitivo para a sua distribuição Linux

  • AUR - pacman -S keepassxc
  • RPM - dnf install keepass
  • DEB - apt-get install keepass2
