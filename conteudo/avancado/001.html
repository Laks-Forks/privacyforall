Sistemas Operacionais de PC

Debian

É um sistema operacional composto inteiramente de software livre. É mantido oficialmente pelo Projeto Debian. O projeto recebe ainda apoio de outros indivíduos e organizações em todo mundo.

Website

Qubes OS

Qubes OS é um sistema operacional com foco em segurança para desktops com o objetivo de prover segurança através de isolamento. O motor de virtualização é o Xen, e os ambientes isolados criados pelo usuário podem ser baseados no Fedora, Debian, Whonix e Windows.

Website

OpenBSD

O OpenBSD é um sistema operacional livre da família UNIX, multiplataforma. Seu time de desenvolvedores enfatiza a portabilidade, padronização, correção e segurança.

Website

Fedora

Fedora é um sistema operacional ou sistema operativo Linux. O sistema operacional Fedora Linux é software livre e de código aberto, e os programas disponíveis dentro de seu repositório de programas também são programas livres que aderem a uma licença livre.

Website

Sistemas Live

Tails

Tails, The Amnesic Incognito Live System, é uma distribuição Linux baseada no Debian que tem como objetivo manter a privacidade, anonimato e segurança do utilizador e burlar a censura de conteúdo em países.

Website

Tiny Core Linux

O Tiny Core Linux é um sistema operacional mínimo baseado em kernel Linux, com foco em fornecer um sistema básico usando o BusyBox e o FLTK, desenvolvido por Robert Shingledecker. A distribuição é notável por seu pequeno tamanho e minimalismo; funções adicionais são fornecidas por extensões.

Website


Não use Windows 10 - Ele é uma ameaça para a sua privacidade

O Windows 10 vem com diversos recursos que em versões anteriores do Windows não tinha, mas a maioria delas violam sua privacidade
  1. Sincronização de dados habilitada por padrão

    • Histórico de navegação.
    • Configurações de aplicativos.
    • Nomes e senhas de redes WiFi.

  2. Identificador único de anúncio

    • Usados por anunciantes externos para mostrar anúncios personalizdos para você

  3. Cortana pode coletar qualquer dado seu

    • As teclas apertadas no teclado, buscas e microfone
    • Seu calendário
    • Músicas que você toca no computador
    • Dados de cartão de crédito
    • Compras

  4. Microsoft pode coletar qualquer dado sobre você

    • Sua identidade
    • Senhas
    • Sua idade
    • Interesses e hábitos
    • Dados de uso
    • COntatos e relacionamentos
    • Dados de localização
    • Emails
    • Mensagens instantâneas
    • Lista de chamadas
    • Áudios
    • Gravações de vídeo

  5. Seus dados podem ser compartilhados

    • Ao usar o Windows 10, você autoriza a Microsoft em compartilhar qualquer um dos dados citados anteriormente por entidades escolhidas por ela.
